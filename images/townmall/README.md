# Town Mall Images

Place item images in this folder with the following filenames:

- **gigrometr.jpg** - Гигрометр (20 coins)
- **plush_toy.jpg** - Рандомная мягкая игрушка (40 coins)
- **board_game.jpg** - Рандомная новая настолка с пиндош (100 coins)
- **humidifier.jpg** - Увлажнитель воздуха (120 coins)
- **tent.jpg** - Двухместная палатка (1000 coins)

## Supported formats:
- JPG/JPEG (recommended)
- PNG
- WEBP
- GIF

## Recommendations:
- Keep images under 5MB for faster loading
- Recommended size: 800x600 pixels or similar aspect ratio
- Use descriptive, clear photos of the items
